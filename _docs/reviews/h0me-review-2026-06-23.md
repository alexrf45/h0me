## Executive Summary

Headline: the **Talos 0.12 migration, Flux Operator bootstrap, and Cilium GitOps
handover all landed and the cluster is up** — the entire 06-22 CRITICAL/HIGH tier
(C1/C2/H1/H2) is resolved. The Flux DAG is green and all platform HelmReleases
(cilium, flux-operator/instance, cnpg, ESO, cert-manager, kyverno, etc.) are Ready.
Storage was the last blocker: democratic-csi was CrashLooping on (a) overlapping
ZFS dataset paths and (b) a stale TrueNAS API key — both fixed; the controller is
now **5/5 Ready** and the statically-provisioned CNPG database PVCs are **Bound**.

What's still broken is downstream of that long outage: **dynamically-provisioned
PVCs (CNPG dump volumes + the whole observability stack) are still Pending**, a set
of **pods from the broken-CSI era are stuck** (Init:0/1 / CrashLoop) and need
recreation, and the **wildcard cert is failing** because the Cloudflare DNS-01
token ExternalSecret won't sync.

Recommended next sprint: **clear the post-storage fallout** — verify/provision the
Pending dynamic PVCs (TrueNAS-side per MN1), recreate the stale pods, and fix the
`cf-token` ExternalSecret so the wildcard cert issues.
